Principais Formaações de Texto

Negrito / destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase temos um termo em destaque utilizando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase, temos um termo em itálico usando a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> (semântica)

Texto Grande e Pequeno

Estamos criando um texto grande utilizando a tag <big> e um texto pequeno utilizando a tag <small> (semântica)

Texto deletado

Podemos marcar um texto como deletado utilizando a tag <del> para indicar que ele deve ser lido mas não considerado (semântica)

Texto inserido

Podemos marcar um texto como inserido utilizando a tag <ins> para dar uma ênfase e indicar que ele foi adicionado depois (semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20 + 3, utilizando a tag <sup>

Texto Subscrito

Para inserir coisas do tipo H2O, utilizando a tag <sub>
